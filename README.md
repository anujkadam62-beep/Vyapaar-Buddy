# 📦 Vyapaar Buddy

An AI-powered Smart Inventory Management System designed to help small businesses efficiently manage inventory, monitor stock levels, visualize business analytics, and receive intelligent alerts through a modern dashboard.

---

## 🚀 Overview

Vyapaar Buddy is a business management application built using **Python** and **Flet**. It provides an intuitive interface for inventory management, business analytics, AI-assisted interaction, and real-time monitoring through a unified dashboard.

The application is designed for retailers and small business owners who want a simple yet powerful solution to manage their inventory and gain insights into their business.

---

## ✨ Features

- 🔐 User Authentication
- 📊 Interactive Business Dashboard
- 📦 Inventory Management
- 🤖 AI Chat Assistant Interface
- 🚨 Smart Alerts & Notifications
- 📈 Sales & Analytics Dashboard
- ⚙️ User Settings
- 📱 Responsive Desktop Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/anujkadam62-beep/Vyapaar-Buddy.git
```

### Move into Project

```bash
cd Vyapaar-Buddy
```

### Install Dependencies

```bash
pip install -r requirmenta.txt
```

### Run Application

```bash
python main.py
```
